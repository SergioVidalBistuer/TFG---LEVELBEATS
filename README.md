## Descripción

**LevelBeats** es una aplicación web desarrollada como proyecto final, enfocada en conectar a productores, ingenieros y usuarios dentro de un mismo ecosistema digital.  
La plataforma permite centralizar la gestión de contenidos musicales y servicios, facilitando la compraventa de beats, la contratación de servicios técnicos y la interacción entre perfiles profesionales.

## Objetivos principales

- Publicación y exploración de **beats** y **colecciones musicales**.
- Contratación de **servicios profesionales** dentro del marketplace.
- Gestión de perfiles con roles como **Productor** e **Ingeniero**.
- Sistema de **compras**, **facturación** y seguimiento de actividad.
- Entorno privado de usuario con herramientas de gestión y analíticas.

## Tecnologías utilizadas

- **Laravel**
- **PHP**
- **Blade**
- **MySQL**
- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap**
- **Docker**

## Funcionalidades destacadas

- Marketplace de beats, colecciones y servicios.
- Sistema de autenticación y registro.
- Inicio de sesión con Google.
- Gestión de roles y planes.
- Carrito, checkout y facturación.
- Paneles de analíticas.
- Sistema de mensajería y contacto.
- Área privada de usuario y entorno Studio.

## Estructura del proyecto

El proyecto sigue una arquitectura basada en **Laravel MVC**, organizando la lógica en:

- **Controladores**
- **Modelos**
- **Vistas Blade**
- **Servicios y helpers**
- **Middleware**
- **Recursos estáticos y estilos**

## Estado del proyecto

Actualmente, **LevelBeats** se encuentra en una fase avanzada de desarrollo y preparación para despliegue, con una estructura refactorizada, documentación interna y organización orientada a una entrega de calidad.
